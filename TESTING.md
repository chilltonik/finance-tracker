# Тестирование системы тем

## Быстрый тест

```bash
# Активировать виртуальное окружение
source venv/bin/activate

# Запустить тестовый скрипт
python3 test_themes.py
```

## Запуск приложения

```bash
source venv/bin/activate
python3 main.py
```

## Известные ограничения

- Изменение темы пересоздает UI (кратковременное мигание)
- Форма Add Transaction теряет данные при смене темы (ожидаемое поведение)
