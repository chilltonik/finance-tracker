/init - создание CLAUDE.md с описанием всего проекта

CLAUDE.md - используется при каждом вызове claude. Является дополнительным контекстом

/permissions - задать права на проекта

/config - можно настроить клод в целом и посмотреть на описание аккаунта и использование лимитов

/model - сменить модель

/status - посмотреть статус
/stats - статистики использования
/usage - лимиты

/memory + prompt - использовать проектную память или пользовательскую. Это факты

alt + t - включение/выключение режима размышления

shift + tab: accept edits on - принимает все без одобрения, plan mode on - сначала план, потом действия (задает уточняющие вопросы, дает итоговый план)
